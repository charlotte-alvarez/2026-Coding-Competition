# 08 - Human Readable Time

## Problem Statement

_Write a function, which takes a non-negative integer (seconds) as input and returns the time in a human-readable format (HH:MM:SS)
    HH = hours, padded to 2 digits, range: 00 - 99
    MM = minutes, padded to 2 digits, range: 00 - 59
    SS = seconds, padded to 2 digits, range: 00 - 59
The maximum time never exceeds 359999 (99:59:59)_


> Note: Remember to fill in 0's. Output should always be in the format of HH:MM:SS (e.g., 00:03:45)

## Input Format: 
- int $0 - 359999$

## Output Format:
- str (HH:MM:SS)
    
## Constraints:
- seconds are $>=0$ and $<=359999

## Sample:

| Input | Output |
| ----- | ---------------------- |
| 359999| 99:59:59               |

None
